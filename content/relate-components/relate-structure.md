

- Relate Structure

---
The main configuration area in Relate is called *Relate Structure*. This is where most of the elements that make up a Relate custom application are stored and maintained. Included in this group are [forms](/relate-components/relate-structure/fields), [option lists](/relate-components/relate-structure/option-lists-and-groups), [wizards](/relate-components/relate-structure/wizards), [merge-reports](/relate-components/relate-structure/merge-reports), [multi-entry reports](/relate-components/relate-structure/multi-entry-reports), [formulas](/relate-components/relate-structure/formulas), [folders](/relate-components/relate-structure/folders) and [end points](/relate-components/relate-structure/end-points). Elements created outside Relate structure are [record types](/relate-components/other-elements-and-functions/record-types-and-categories), [record categories](/relate-components/other-elements-and-functions/record-types-and-categories), [bundles](/relate-components/other-elements-and-functions/bundles), [reports](/relate-components/other-elements-and-functions/queries-reports) and [queries](/relate-components/other-elements-and-functions/queries-reports). Also included in Relate, but not in Relate Structure, are a few special functions: [Relate administrators](/relate-components/other-elements-and-functions/relate-administrators), [data import](/relate-components/other-elements-and-functions/data-import), [form import](/relate-components/other-elements-and-functions/form-import) and [create record](/relate-components/other-elements-and-functions/record-creation).Relate structure normally displays all existing Relate elements in a large hierarchical folder tree. Each folder can be expanded and collapsed to hide or show the group of elements contained in the folder. Relate structure can also be filtered to show only certain types of elements. If the filter is set to not show folders, then any remaining items are displayed in a big list without the hierarchal folder structure. This can be very useful if you can't remember where you put something. The filter is configured at the top of the Relate structure page.There are a lot of icons on the Relate structure page. You can find the meaning of any icon by holding your mouse pointer over it. A tool tip will pop-up to describe the icon. Also, at the bottom of Relate structure is a legend that list all icons and their meanings.

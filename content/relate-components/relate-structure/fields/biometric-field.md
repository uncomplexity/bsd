

- Biometric Field

---
Biometric fields interact with special biometric hardware devices: signature pads and fingerprint readers. Obviously the hardware device must be present on the user's computer to take advantage of these fields. The data recorded includes biometric data from the hardware device as well as a timestamp and optional hash value. This data is described in more detail in the [RelateScript documentation](https://bluestepplatformsupport.bluestep.net/shared/custompage/custompage.jsp?_event=view&_id=445506_U129801__8501). Below is a description of the settings unique to biometric fields. Other settings are described in the parent article on fields.

| **Setting** | **Description** |
| --- | --- |
| Format | This setting determines what type of hardware the field interacts with. Fingerprint devices come in two varieties depending on which version of the driver software supports that device. |
| Timestamp | Determines whether the stored timestamp is displayed, and if so in what format. |



- Data Types

---
This section contains a brief desciption of the basic types of data supported by Relate Script. The basic data types include String, Integer, Float, Boolean, Date, Time, DateTime, DateDiff and Array. There are also a number of special types used to represent various kinds of Relate data, but these types are discussed in their own section: [Working with Relate Data](/relate-script/working-with-relate-data).

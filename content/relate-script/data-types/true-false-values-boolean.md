

- True/False Values: Boolean

---
Boolean values are the simplest of all values. They can be true or false. Yet, with just these two values the entire digital age was born. For a computer, every decision and every computation must come down to a simple true/false, yes/no or 1/0 and that is where Boolean values come in. In Relate Script there is a third possible value, null, meaning unknown or no data.There are many ways to work with Boolean values, all of which are discussed in the section on operators.

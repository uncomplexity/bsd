

- drugFoodByProductID(...)B.Drug.foodInteractions

---
The drugFoodByProductID(...) function returns foods that may interact with a drug. The result of drugDrugByProductID(...) is a list of possible food interactions in an array of type String[Integer] where the first value is at index 0:**Syntax:**
**drugFoodByProductID**( productID ) productID.**drugFoodByProductID**( )

| **Parameter** | **Description** |
| --- | --- |
| *productID* | ProductID for a drug from the "medName" lookup |

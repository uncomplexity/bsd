

- size(...)

---
The size(...) function has only one variation, not counting syntax variations. It computes the number of elements in an Array value. If the Array value is null the result is null.**Syntax:**
**size(** **array-value** **)**
*array-value***.size()**

| **Parameter** | **Description** |
| --- | --- |
| *array-value* | The array value whose size is being computed. |

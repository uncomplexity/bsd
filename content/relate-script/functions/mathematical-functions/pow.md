

- pow(...)

---
The pow(...) function has only one variation, not counting syntax variations. It raises a number to the power of another number. This operation is commonly called y to the x. If both inputs are Integer values the result is an Integer, otherwise the result is a Float value. If either input is null the result is null.**Syntax:**
**pow(** **number***,** *exponent** **)**
*number***.pow(** *exponent* **)**

| **Parameter** | **Description** |
| --- | --- |
| *number* | The number to be raised to a power. |
| *exponent* | The power by which *number* will be raised. |

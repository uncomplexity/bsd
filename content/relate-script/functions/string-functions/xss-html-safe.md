

- xssHtmlSafe(...)

---
Take a string of HTML and XSS proof it. XSS, or cross-site scripting, is when someone attacks you by inserting malicious code into a field where you accept user input. It's a good practice to sanitize your inputs. Any Relate-built inputs will get XSS filtered on submit. **Syntax:** xssHtmlSafe(string)

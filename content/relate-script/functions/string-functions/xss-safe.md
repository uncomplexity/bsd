Take a string and XSS proof it. XSS, or cross-site scripting, is when someone attacks you by inserting malicious code into a field where you accept user input. It's a good practice to sanitize your inputs. If you are accepting HTML, use `xssHtmlSafe()`. **Syntax:** xssSafe(source-string)



- Accessing Forms and Fields

---
This section describes how forms and, by extension, fields are accessed. It does not describe how to access the data actually contained in the fields, only how to get to the fields themselves in a generic way. For a description of how to access the different types of data contained in each type of field, see the previous section on accessing [Field Data](/relate-script/working-with-relate-data/field-data).The following sections are arranged progressively from simplest to most complex beginning with accessing fields and ending with accessing data from a query or report.

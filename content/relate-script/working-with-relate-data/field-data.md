

- Field Data

---
This section describes the individual attributes of different types of fields in Relate. It describes how to access and modify the data contained in the more complex fields types such as multi-select and signature fields.. This section does not describe how to access the fields themselves, only how to access and change the data within the fields. For more information on how fields themselves are accessed see the next major section titled [Accessing Forms and Field](/relate-script/working-with-relate-data/accessing-forms-and-fields). There are 8 types of fields on Relate forms with data that can be access and modified. Additionally there is a pseudo-field on every form which allows read-only access to information stored internally by BlueStep, such as who last modified the data and when.
